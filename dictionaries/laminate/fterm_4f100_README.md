# 4F100 (積層体) Fターム辞書 — 利用上の注意

## このファイルは何か

`fterm_4f100_structure.json` は本リポジトリで `field=laminate` の案件に対して
Fターム候補を提示するための辞書データです。cosmetics 分野 (4C083) と同じ目的で
使われ、`services/keyword_service.py::fterm_candidates()` 経由で Step 3 の
「Fターム候補」パネルに表示されます。

## 精度レベル / 由来

- **version 0.2.0-ai-draft**: Claude (AI) が JPO の 4F100 分類表の一般的な
  構造知識に基づいて起草した拡張ドラフト版です。約 170 項目。
- **バージョン 0.1** (22 項目のみのスケルトン) から約 8 倍に拡張済み。
- **精度は J-PlatPat 公式分類表より劣ります**。特に以下に注意：
  - **コード番号 (例: AK01, AK03)** は一般的な番号域に従って推定したもので、
    一部は JPO の正式な付与番号と一致しない可能性があります。
  - **ラベル (日本語説明)** と **examples (例示語)** は比較的信頼できます。
  - **カテゴリ (AA/AK/BA/GB/JK/YY 等)** の存在自体は公式。

## 推奨ワークフロー（重要）

1. **本ツール内で「候補を探す」**
   - Step 3 の Fターム候補パネルで、検索ボックスに製品の特徴を入力（例：
     `EVOH`, `ポリエステル`, `バリア`, `食品包装` 等）。
   - 該当候補（ラベル・例示）を眺める。
2. **J-PlatPat で正式なコードを確認**
   - `https://www.j-platpat.inpit.go.jp/` → PMGS (特許分類照会) → Fターム検索
   - テーマコード `4F100` で分類表を開き、該当ラベルの**正式なコード**を特定。
3. **正しいコードで検索式を組む**
   - 本ツール内でも手動で任意の Fターム コードを追加できます（Step 3 の各
     グループの「コード追加...」）。正式コードを確認後、上書き追加してください。

## 辞書の拡充・修正方法

この JSON ファイルは人間が直接編集できる構造です。

### 構造

```jsonc
{
  "theme_code": "4F100",
  "theme_name": "積層体",
  "version": "0.2.0-ai-draft",
  "categories": {
    "AK": {                              // 観点コード (2文字)
      "label": "層の材質",                // 観点の日本語ラベル
      "entries": {
        "AK01": {                         // 具体的なFタームコード
          "label": "ポリオレフィン",        // 日本語説明
          "examples": ["PE", "PP"]         // 例示語 (逆引き用)
        }
      }
    }
  }
}
```

### 新しい項目を追加する

1. 上記 `categories.[観点].entries` に新エントリを追記。
2. コード番号は重複しないように注意（同じ観点内で一意）。
3. `label` は簡潔に、`examples` には 2〜5 個の代表的な語を入れるのが目安。
4. アプリを再起動すれば `@lru_cache` がリセットされ新データが読まれます
   （`modules/fterm_dict.py::_load_json` にキャッシュがあるため）。

### ロード形式（内部動作）

アプリ起動時、`modules/fterm_dict.py::_normalize_structure_to_tree()` が
このファイルを `{nodes, reverse_index}` 形式に自動変換します。Cosmetics の
`fterm_4c083_tree.json` と同じツリー API で扱えるようになります。
テストは `tests/test_fterm_dict.py` にあります。

## 公式データ源（推奨読み込み元）

将来、AIドラフトから公式データに置き換える場合の参考：

- **特許庁 PMGS**: <https://www.j-platpat.inpit.go.jp/p0100> (分類照会)
- **特許庁 Fターム付与基準書 (Theme Code 4F100)**: JPO 公式の PDF / CSV
- **テーマコード 4F100 の正式名称**: 積層体 (Laminate)

公式 CSV を取得できた場合は、`tools/` 配下に変換スクリプトを追加して
`fterm_4f100_tree.json` (tree 形式) として保存することを推奨します。
